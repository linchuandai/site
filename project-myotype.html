MyoType

This is a project that I worked on in a team of four at EC Hacks. We made this project as a way for people to type without the use of a keyboard. It was designed for people for who do have fine motor control and can only make "large" movements. This project utilizes the Myo Arm Band, which can track the position of your arm, along with certain actions of the hand. By using the tracking technology of the arm band, we were able to create a system where the user would be able to control a tree and select certain values which would correspond to letters, spaces, punctuation, etc...

The specific controls are:
  • Moving left and right down the binary tree
  • Having a backspace button that deletes the last character
  • Choosing the current character where the cursor points
  • Changing the speed at which you traverse down the tree

For this project, the major challenge that we ran into was translating what the sensors was reading into information that we could use to translate these into certain actions. From the sensors, we could gather information such as change in pitch, yaw, and roll, which we could then translate into twisting, and lateral movement in the different directions.
